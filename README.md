  - Python
- データベース
  - MySQL 

## どんなアプリ？
「日記が長続きしない人」や「最近の自分について知りたい人」向けの日記アプリです。
- 日記を書くことで木が成長するため、楽しく日記を続けることができる
- 木を見ることで、最近の自分の言葉遣いが分かり、結果的に自己を振り返ることができる
